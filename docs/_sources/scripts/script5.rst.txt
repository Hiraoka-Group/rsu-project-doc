Script 5
==============
This script reproduces the following figure from the paper:

RSU vs. theta plot for the following rings:

- syn-S-2
- syn-S-1
- syn-T-1
- minimum RSU of dimeric rings

Usage
--------

Before running the script, make sure your current working directory is the `reprod` directory. 

Then run the following command:

.. code-block:: bash
   
   python script5.py

--------
Another window will pop up, showing a plot like the following:
