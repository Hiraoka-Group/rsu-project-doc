Script 1
==============
This script reproduces the following figure from the paper:

RSU vs. theta plot for syn-S-1 with :math:`\delta` = 87° and 90°.

Usage
--------

Before running the script, make sure your current working directory is the `reprod` directory. 

Then run the following command:

.. code-block:: bash
   
   python script1.py
