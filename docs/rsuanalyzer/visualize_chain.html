visualize_chain#

visualize_chain ·#

rsuanalyzer.visualize_chain.visualize_chain.visualize_chain(conf_id: str, theta: float, delta_: float = 87, show: bool = True) → tuple[Figure, Axes3D]#

Visualize the structure of a chain.

You can either show the plot or get the figure and the axes object. The latter is useful when you want to customize the plot. (See Example)

Caution

This function is intended to provide an approximate visualization of the chain structure. It may not accurately represent the precise positions of the atoms.

Parameters:

  • conf_id (str) – The conformation ID of the chain, e.g. “RLFFRLFFRL”. Ring ID (e.g. “RLFFRLFFRLFF”) is also acceptable, but the last two characters will be ignored.

  • theta (float) – The tilt angle of the C-C bonds in degrees.

  • delta (float, optional) – The N-Pd-N angle in degrees. Default is 87.

  • show (bool, optional) – If True, the plot will be shown. Default is True. Set False if you just want to get the figure and the axes object.

Returns:

A tuple of the figure and the 3D axes object.

Return type:

tuple[Figure, Axes3D]

Example

Case 1:

Plot the structure of a chain.

>>> import rsuanalyzer as ra
>>> ra.visualize_chain("RLFFRLFFRLFF", 34)
Case 2:

Get the figure and the axes object. This is useful when you want to customize the plot.

>>> import matplotlib.pyplot as plt
>>> import rsuanalyzer as ra
>>> fig, ax = ra.visualize_chain("RLFFRLFFRLFF", 34, show=False)
>>> ax.set_title("syn-T-1, theta=34, delta=87")
>>> plt.show()

carbons#

rsuanalyzer.visualize_chain.carbons.calc_carbon_positions(conf_id: str, theta: float, delta_: float = 87) → list[list[list[ndarray]]]#

Calculate the positions of the carbon atoms in the ligands of a chain.

This function is intended to be used for visualizing chains with matplotlib 3D plots. The calculated positions of the carbon atoms are represented in the global coordinate system.

Caution

This function is intended to provide an approximate visualization of the ligand structure. It may not accurately represent the precise positions of the atoms.

Hint

You can use visualize_chain to visualize chains more easily.

Parameters:

  • conf_id (str) – The conformation ID of the chain, e.g. “RRFFLL”.

  • theta (float) – The tilt angle of the C-C bonds in degrees.

  • delta (float, optional) – The N-Pd-N angle in degrees. Default is 87.

Returns:

A list of lists of lists of numpy arrays. Each list corresponds to a ligand in the chain. Each inner list corresponds to a fragment of the ligand molecule. The numpy arrays in the innermost lists represent the positions of the carbon atoms in the global coordinate system.

Return type:

list[list[list[np.ndarray]]]

Example

>>> import matplotlib.pyplot as plt
>>> import rsuanalyzer as ra
>>> fig, ax = plt.subplots(subplot_kw={'projection': '3d'})
>>> c_positions_of_ligs = ra.calc_carbon_positions("RLFFRLFFRL", 34)
>>> for lig in c_positions_of_ligs:
...     for frag in lig:
...         xs = [vtx[0] for vtx in frag]
...         ys = [vtx[1] for vtx in frag]
...         zs = [vtx[2] for vtx in frag]
...         ax.plot(xs, ys, zs)
>>> ax.set_xlim(-.2, 1.8)
>>> ax.set_ylim(-.25, 1.75)
>>> ax.set_zlim(-1, 1)
>>> ax.set_box_aspect([1, 1, 1])
>>> plt.show()

metals#

rsuanalyzer.visualize_chain.metals.calc_metal_positions(conf_id: str, theta: float, delta_: float = 87) → list[ndarray]#

Calculate the positions of the metal atoms in a chain.

Caution

This function is intended to provide an approximate visualization of the chain structure. It may not accurately represent the precise positions of the atoms.

Hint

You can use visualize_chain to visualize chains more easily.

Parameters:

  • conf_id (str) – The conformation ID of the chain, e.g. “RLFFRLFFRL”.

  • theta (float) – The tilt angle of the C-C bonds in degrees.

  • delta (float, optional) – The N-Pd-N angle in degrees. Default is 87.

Returns:

A list of numpy arrays representing the positions of the metal atoms in the global coordinate system.

Return type:

list[np.ndarray]

Example

>>> import matplotlib.pyplot as plt
>>> import rsuanalyzer as ra
>>> fig, ax = plt.subplots(subplot_kw={'projection': '3d'})
>>> metal_positions = ra.calc_metal_positions("RLFFRLFFRL", 34)
>>> xs = [vtx[0] for vtx in metal_positions]
>>> ys = [vtx[1] for vtx in metal_positions]
>>> zs = [vtx[2] for vtx in metal_positions]
>>> ax.scatter(xs, ys, zs)
>>> ax.set_xlim(-.2, 1.8)
>>> ax.set_ylim(-.25, 1.75)
>>> ax.set_zlim(-1, 1)
>>> ax.set_box_aspect([1, 1, 1])
>>> plt.show()
