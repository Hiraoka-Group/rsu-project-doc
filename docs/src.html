Module contents#

This package provides a set of tools for analyzing RSU (Ring Strain Unit).

The package includes the following submodules:

  • calc_rsu: A set of tools for calculating RSU.

  • analyze_rsu: A set of tools for analyzing RSU.

  • enum_ids: A set of tools for enumerating conformation IDs of rings.
